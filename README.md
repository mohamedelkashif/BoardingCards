# BoardingCards
 Installation:
 ```
 composer install
 ```
 
 If you don't have composer, you can get it from  https://getcomposer.org/ .<br />
  Open cmd and and navigate to the project folder then type 
 ```
 composer install
 ```
 
 <br /> Or 
 ```
 composer update
 ```
## Run the application
 
```
php index.php
```
It should output an ordered list of the boarding cards
